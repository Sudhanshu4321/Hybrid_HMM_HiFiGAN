System Description (For Hindi Dataset)

System 2: HMM-based TTS System trained on read-speech using mgc as spectral feature
System 3: Hybrid HMM model-HiFi-GAN model using Histogram Equalization
System 4: FastSpeech model trained on read speech
System 5: HMM-based TTS System trained on read-speech using 80 mel-specs as spectral feature
System 6: HMM-based TTS System trained on read-speech using 120 mel-specs as spectral feature
